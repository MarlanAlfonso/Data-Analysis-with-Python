## What This Certification Covers

This certification validates practical skills in Python-based data analysis — loading and cleaning real-world datasets, computing statistical summaries, and producing professional data visualizations. It is designed for developers and analysts who work with structured tabular data and need to extract insights programmatically rather than through spreadsheet tools.

> "This certification covers the full data analysis workflow in Python — from reading raw CSV files and handling missing or invalid data, to computing descriptive statistics and producing publication-quality charts. It uses the same libraries — NumPy, Pandas, Matplotlib, and Seaborn — that professional data analysts use daily."

### Modules / Topics Covered

- Part 1: Introduction to Data Analysis — the data analysis process, Jupyter Notebooks, reading data from CSV, Excel, SQL, HTML, and JSON sources
- Part 2: NumPy — array creation, indexing, reshaping, vectorized operations, broadcasting, boolean masking, and linear algebra
- Part 3: Pandas Series and DataFrames — label-based indexing, conditional filtering, groupby aggregation, and column operations
- Part 4: Data Cleaning — handling missing values, invalid data, domain-specific errors, deduplication, and visualization-assisted cleaning
- Part 5: Data Visualization — Matplotlib object-oriented API, Seaborn statistical charts, and choosing the right chart type for the data


## Relevance to My Work

> "I pursued this certification to build a solid foundation in data analysis as a 3rd-year Computer Science student. The projects in this certification gave me hands-on experience working with real datasets — US Census data, medical examination records, environmental data, and web traffic logs — which directly mirrors the kind of work done in data engineering, analytics, and backend development roles.
