## Project Overview

| # | Project | Description | Tech Used | Status |
|---|---------|-------------|-----------|--------|
| 1 | [Mean-Variance-Standard Deviation Calculator](./projects/Mean-Variance-StandardDevia...) | Computes mean, variance, standard deviation, max, min, and sum across rows, columns, and the full 3x3 matrix from a 9-element input list | Python, NumPy | Complete |
| 2 | [Demographic Data Analyzer](./projects/Demographic-Data-Analyzer) | Answers 9 demographic and economic questions from 32,000 US Census records — including education-income relationships and country-level earnings rates | Python, Pandas | Complete |

| 5 | [Sea Level Predictor](./projects/Sea-Level-Predictor) | Fits two linear regression models to EPA sea level data — one from 1880 and one from 2000 — and extends both predictions to 2050 to compare rate scenarios | Python, Pandas, Matplotlib, SciPy | Complete |

## Skills Demonstrated

**Data Manipulation**
- Reshaping and aggregating data with Pandas — groupby, pivot, melt, merge
- Matrix operations and vectorized computation with NumPy
- Loading and parsing CSV files with datetime indexing and type conversion

**Statistical Analysis**
- Descriptive statistics across axes — mean, variance, standard deviation, min, max, sum
- Percentile-based outlier detection and removal
- Linear regression with SciPy for trend analysis and prediction

**Data Visualization**
- Line charts, bar charts, scatter plots, histograms, and box plots with Matplotlib
- Categorical plots and annotated correlation heatmaps with Seaborn
- Multi-panel figures and time series visualizations
